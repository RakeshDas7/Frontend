Higher Order Functions

when a function takes a function as an arguement or return a function as a value then that function called as higher order function

Callback Function

- When A is passed as an arguement within another function then that function called as callback function

Ex -
function A(cb) {}
function B() {}
A(B)
here A -> Higher Order function
here B -> Callback function
